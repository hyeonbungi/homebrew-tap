# homebrew-tap

Personal [Homebrew](https://brew.sh) tap for [hyeonbungi](https://github.com/hyeonbungi)'s tools.

## Usage

```sh
brew install hyeonbungi/tap/sync-ai-clis
```

## Formulas

| Formula | Description |
| --- | --- |
| `sync-ai-clis` | Detect, install, and keep up to date multiple AI coding CLIs (Claude Code, Codex, Gemini, Kiro, Antigravity) with one command |

Formulas in this tap are generated and pushed automatically by [dist](https://github.com/axodotdev/cargo-dist) from each project's release workflow — please don't edit them by hand. Issues belong on the tool's own repository.
